# docs/llm_prompts.md — LLM Prompt-шаблоны

## evaluation.yaml

Располагается: `src/agents/prompt_templates/evaluation.yaml`

### Назначение
Первичная оценка объявления о продаже квартиры. LLM анализирует параметры квартиры,
рыночные условия и выносит вердикт: **hot** / **warm** / **cold** / **reject**.

### Структура

```yaml
system: "..."           # Системный промпт — роль и правила
user_template: "..."    # Шаблон с {{variables}}
response_schema: { ... } # JSON-схема ответа
```

### Переменные user_template

| Переменная | Пример | Описание |
|------------|--------|----------|
| `{{city}}` | Москва | Город |
| `{{district}}` | Пресненский | Район |
| `{{rooms}}` | 1 | Комнат |
| `{{total_area}}` | 46.5 | Общая площадь |
| `{{price}}` | 12000000 | Цена |
| `{{price_per_m2}}` | 258065 | Цена за м² |
| `{{metro}}` | Деловой центр | Ближайшее метро |
| `{{building_year}}` | 2020 | Год постройки |
| `{{object_type}}` | secondary | Тип: secondary / new |
| `{{developer}}` | PIK | Застройщик (если новостройка) |
| `{{repair_type}}` | cosmetic | Тип ремонта |
| `{{floor}}` | 12/25 | Этаж |

### Возможные вердикты

| Вердикт | Score-диапазон | Действие |
|---------|---------------|----------|
| **hot** | 70–100 | Мгновенное уведомление в Telegram |
| **warm** | 50–69 | Пометить для перепроверки через 2 часа |
| **cold** | 30–49 | Долгая перепроверка (2 часа × 5) |
| **reject** | 0–29 | Игнорировать |

### Критерии оценки (примерные)

- **Цена/м² vs рынок** — ниже рынка на 15%+ → сильный плюс
- **Локация** — центр, рядом метро → плюс
- **Ремонт** — после ремонта → плюс
- **Застройщик** — reputable (PIK, Samoleto, etalon) → плюс
- **Год постройки** — новый дом → плюс для вторички
- **Этаж** — не первый, не последний → плюс

### Параметры LLM-вызова

```python
model="qwen36"
temperature=0.3          # Детерминированный ответ
max_tokens=8000          # С запасом (избежать truncation)
response_format={"type": "json_object"}  # JSON mode
```

### Формат ответа (JSON)

```json
{
  "score": 72,
  "verdict": "hot",
  "reasons": [
    "Цена на 20% ниже рынка для района",
    "Новый дом 2023 года, косметический ремонт",
    "5 мин пешком до метро"
  ]
}
```
